
## Download
```
wget https://raw.githubusercontent.com/uGeek/todo.txt-sync/master/todo.txt-sync
```

## Run with screen
```
screen -dmS todo.txt-sync todo.txt-sync.sh
```

## With cron

```
@reboot ( screen -dmS todo.txt-sync /home/angel/todo.txt-sync/todo.txt-sync.sh )
```
